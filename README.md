# Oil-Region-Profitability-Analysis

**Objective:** Develop a machine learning model to identify the most profitable region for opening 200 new oil wells based on predicted oil reserves and risk analysis.
The project aims to:

- Predict the reserve volume of new oil wells.
- Select the top-performing wells from each region.
- Estimate potential profits under a fixed development budget.
- Evaluate financial risk using bootstrapping simulations.
- Choose the region with the highest expected profit and acceptable risk level.

**Project Tasks:**
- Train a Linear Regression model to predict oil reserve volumes for new wells.
- Select the 200 wells with the highest predicted reserves from 500 sampled locations.
- Calculate expected profits considering.
- Perform risk assessment using bootstrapping.
- Identify the region with the highest average profit and a risk of loss below 2.5%.

**Problem Type:**
- Linear Regression
- Confidence Interval
- Bootstrapping

**Evaluation Metrics:**
- RMSE (Root Mean Squared Error)
- Total Profit
- Average Profit
- Risk of Loss
